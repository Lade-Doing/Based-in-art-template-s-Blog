# 博客管理系统
## 介绍
博客项目分为前台和后台，前台分为文章列表页面、文章详细页面、登录页面，后台分为用户管理模块、
文章管理模块。该系统采用B/S架构，使用JWT进行身份验证。
## 技术栈
前台：html、CSS、JavaScript、express-art-template

后台：node.js、express、

数据库：MongoDB

## 环境
1. node16.18.1以上
2. npm9.6.1以上
## 安装依赖
    npm install
## 启动项目
    node app.js

## 注意

前往`model/connect.js`先进行数据库配置
